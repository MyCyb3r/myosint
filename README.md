# MyOsint 
  MyOsint is an integration of maigret sherlock and mr.holmes.

  For Debian based system.

  Installation:

  * git clone https://github.com/MyCyb3r/myosint ~/myosint && cd ~/myosint
  * chmod +x install.sh
  * ./install.sh

  Usage:

  * myosint
